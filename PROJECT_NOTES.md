# 77-441 Reunion Project Notes

## Project Purpose

This project is a reunion webpage for classmates from the Radar major at Nanjing University of Science and Technology.

Current theme:
- Chinese-language reunion page
- Class number shown as `77-441`
- Radar-inspired visual design
- Green radar screen effect

## Main Files

- `index.html`: page content and structure
- `styles.css`: colors, layout, and visual styling
- `script.js`: small dynamic text updates

## Publishing

This project is intended to be published with GitHub Pages from the repository for this site.

Typical publish flow:
1. Edit files locally
2. Review changes
3. Commit and push to GitHub
4. GitHub Pages updates the public site

## How To Make Small Changes

- Change visible text in `index.html`
- Change colors, spacing, sizes, and layout in `styles.css`
- Change simple click/show behaviors in `script.js`
